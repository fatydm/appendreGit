    4. Traduis avec tes mots ce message d’erreur. Quelles prochaines actions dois-tu faire ?

    Auto-merging index.html CONFLICT (content): Merge conflict in index.html Automatic merge failed; fix conflicts and then commit the result.


    Il y a un conflit d'intérêt entre les deux documents. L'ordinateur ne sait pas ce qu'il doit enregistrer. On doit Faire :

  • git config pull rebase false
    • git pull
  • git commit
  • git push
